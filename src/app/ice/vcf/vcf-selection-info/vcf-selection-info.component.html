Variant File Upload

The Variant Call Format (VCF) specifies the format of a text file used in bioinformatics for storing variation data. It is used by large scale variant mapping projects such as The International Genome Sample Resource (IGSR). It is also the standard output of variant calling softwares such as GATK, BCFTOOLS, VArscan FREEbays etc. VCF is a preferred format because it is unambiguous, scalable and flexible, allowing extra information to be added to the info field. Many millions of variants for population can be stored in a single VCF file. Please Upload your vcf file for Storage / Analyse / Query / Visualise
