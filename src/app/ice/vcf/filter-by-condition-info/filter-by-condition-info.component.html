Variant Prioritization

VCF carry so many informations about genomic variants. The info column includes position-level information for that data row and can be thought as aggregate data that includes all of the sample-level information specified such as AC, AF, AN, DP, NS etc. These specified information can be used for filtering / Querying the VCF file on basis of either AF or DP value cut-off. If genotype information is present, then the same types of data must be present for all samples. If info field missing the information of AC, AF, AN and DP then using genotype field these information can be updated in info field on user choice. User can can set these values for for customised population set
