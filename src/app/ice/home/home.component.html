Big Data in Biology

Bottleneck in genome sequencing is no longer data generation-but the challenges are around data analysis, display and integration
  • Data analysis: Computational tools are becoming inadequate to handle the volume of data
  • Data integration: Data types like molecular, phenotypic, clinical etc must be linked and correlated to derive models.
  • Visualization: Multidimensional data from different studies must be correlated and visualized
  • Computational tools: Computational infrastructure (storage, CPU power, networks) is required along with tools for rapid analysis
  • Training: Scientists with expertise in biology and informatics, containerr, mathematics, statistics are required.

Anvaya NGS

AnvayaNGS is an automated workflows, with user friendly UI and Predefined pipelines to carry out analysis of NGS data.It also provides drag and drop facility to build custom workflows. Rule Engine incorporated.Can use a backend parallel supercontainerrs for rapid execution.
Key Features
  • Create, Save, Run, Stop, Re-run workflow
  • Rules Engine to ensure logical connectivity of tools
  • Provision of Custom Tools and Custom Parsers
  • 13 Pre-defined Workflows for frequently used pipelines in genome annotation and comparative genomics
  • Easy to use, standalone Anvaya Client which is supported on Windows as well as Linux
